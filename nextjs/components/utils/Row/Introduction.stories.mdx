# The Row component

The `Row` component renders its children in a responive manner by placing them in a set of columns. You should note that only direct children
are responsively rendered. So you must use it as follows..

```jsx
<Row {...props}>
  <Child1 />
  <Child2 />
  <Child3 />
  <Child4 />
  <ChildN />
</Row>
```
